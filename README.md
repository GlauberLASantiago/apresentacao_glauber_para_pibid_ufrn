# 🎓 Palestra: Do Professor-Pesquisador ao Professor-Criador

A **Palestra: Do Professor-Pesquisador ao Professor-Criador** é uma apresentação web interativa desenvolvida para organizar, apresentar e compartilhar reflexões sobre o uso de tecnologias e artefatos digitais na educação musical. O projeto transforma uma palestra em uma experiência navegável, visual e dinâmica, facilitando a exploração dos temas por meio de seções estruturadas, navegação lateral e recursos interativos.

## ✨ Funcionalidades

- Estrutura em formato de apresentação web com múltiplas seções
- Navegação lateral fixa para acesso rápido aos tópicos
- Animações de entrada conforme o conteúdo aparece na tela
- Destaque automático da seção ativa no menu
- Cartões interativos com links para recursos externos
- Botões para:
  - abrir links em nova aba
  - copiar links para a área de transferência
- Tooltips explicativos em elementos específicos
- Layout responsivo para diferentes tamanhos de tela
- Interface visual moderna com rolagem suave

## 📚 Temas abordados

A apresentação percorre discussões sobre:

- a transição do professor-pesquisador para o professor-criador;
- o uso do modelo **TPACK** na educação musical;
- os eixos pedagógicos da educação musical;
- a disciplina **RTEM** e seu percurso formativo;
- ferramentas digitais como **MuseScore**, **Audacity**, **Scratch** e **Makey Makey**;
- o papel da **IA generativa** na criação pedagógica;
- a produção de **artefatos educacionais digitais**;
- o projeto final **Sinfonia Educacional Tecnológica**;
- a integração entre pedagogia, música e tecnologia.

## 🎯 Objetivo do Projeto

O projeto foi criado para apresentar uma palestra de forma mais envolvente e interativa, transformando conteúdos expositivos em uma experiência digital organizada e acessível. A proposta é valorizar não apenas o conteúdo da fala, mas também o design da apresentação como parte do processo pedagógico.

## ▶️ Como usar

1. Abra o arquivo HTML em um navegador moderno.
2. Utilize o menu lateral para navegar entre os slides.
3. Role a página para acompanhar a sequência da apresentação.
4. Clique nos cartões de recursos para acessar materiais complementares.
5. Use os ícones dos cartões para abrir ou copiar links rapidamente.

## 🧩 Estrutura da Apresentação

A palestra está organizada em 12 partes:

1. Título
2. Construção da palestra
3. Fundamento conceitual
4. Disciplina RTEM
5. Ferramentas essenciais
6. Ferramentas de exploração e interatividade
7. IA generativa
8. Artefatos digitais
9. Projeto final

11. Síntese da jornada
12. Contato e recursos adicionais
